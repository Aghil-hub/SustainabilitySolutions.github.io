# SustainabilitySolutions

An interactive web dashboard that compares universities on key sustainability metrics such as energy efficiency, waste reduction, transportation, and campus initiatives.

## Live site

- GitHub Pages: https://aghil-hub.github.io/SustainabilitySolutions.github.io/

## Purpose and context

This dashboard is designed to provide senior leaders and sustainability stakeholders at UIUC with a strategic view of the university’s position in the Times Higher Education (THE) Impact Rankings by:

- Integrating key Impact Ranking metrics, including overall score, rank, SDG-specific scores, number of related research outputs, year-on-year changes in ranking, and associated university funding levels.
- Benchmarking UIUC against top-performing U.S. institutions to illuminate where peer universities are excelling and to identify concrete areas where UIUC can learn from their practices.
- Highlighting Sustainable Development Goals (SDGs) and indicator-level criteria where targeted investments and initiatives are most likely to improve UIUC’s future performance.
- Linking ranking outcomes to resource allocation and strategic priorities, enabling a more evidence-based dialogue between academic, sustainability, and financial decision makers.

Providing a forward-looking, scenario-style view of how UIUC’s standing could evolve if the university elects to participate in the THE Impact Rankings and pursues specific SDG-focused interventions.

## Features

- Radar chart comparing universities across multiple sustainability dimensions.
- Dynamic data loading for different institutions.
- Clean, responsive layout built with HTML, CSS, and JavaScript.
- Visual assets stored in the `images/` folder.

## Project structure

- `index.html` – Main landing page and layout.
- `styles.css` – Styling for the dashboard.
- `script.js` – Logic for charts and interactivity.
- `images/` – Logos and other image assets.

## Getting started (local)

1. Clone the repository:  
   `git clone https://github.com/Aghil-hub/SustainabilitySolutions.github.io.git`
2. Open `index.html` in your browser, or use a simple local server (for example in VS Code with Live Server).

## Development

- Update content and layout in `index.html`.
- Adjust styles in `styles.css`.
- Modify chart logic and data in `script.js`.
- Commit and push to `main` to trigger GitHub Pages deployment.
